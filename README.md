Data Structures course at UC Berkeley

- Classes
- Linked Lists
- Unit Tests
- Subtype Polymorphism
- Exceptions, iterators, iterables
- Asymptotics
- Disjoint Sets
- Hashing
- Heaps
- Graph Traversal (DFS, BFS, Dijkstra, A*)
- Sorting Algorithms (quicksort, heapsort, mergesort)
